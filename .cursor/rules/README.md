# Cursor Rules

This directory contains project rules for the coding agent.

## Domain Structure

- `architecture/` - architecture and layering constraints.
- `react/` - React and UI composition/performance rules.
- `typescript/` - TypeScript authoring conventions.
- `rules/` - meta-rules describing how to write and maintain rules.

## Current Rules

- `architecture/fsd.mdc` - Feature-Sliced Design and segment placement.
- `react/component-composition.mdc` - component composition and render boundaries.
- `react/stable-constants-and-references.mdc` - stable references and module-level constants.
- `typescript/interface-first.mdc` - interface-first typing style.
- `rules/organization-and-naming.mdc` - rule organization and naming conventions.

## Maintenance

When adding, renaming, moving, or removing rule files in `.cursor/rules`, update this `README.md` in the same change.
